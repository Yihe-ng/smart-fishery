# 石斑鱼智慧监测系统 - 用户使用手册

本系统是专为工厂化循环水养殖（RAS）设计的智慧化管理平台，集成了水质监测、AI病害识别、精准投喂和生产数据分析等核心功能。

## 1. 监测大屏 (Dashboard)
大屏首页提供养殖池的实时全景监控：
- **左侧面板**:
  - **水质监测**: 实时显示水温、pH值、溶解氧、氨氮、亚硝酸盐等5项核心指标。正常状态显示绿色，预警显示黄色，告警显示红色。
  - **设备状态**: 监控所有在线传感器的连接状态。
  - **实时告警**: 自动推送异常指标告警，支持一键查看详情。
  - **投喂控制**: 快速配置投喂参数并查看今日投喂日志。
- **右侧面板**:
  - **健康总览**: 基于AI识别结果的鱼类综合健康评分及风险等级。
  - **视频监控**: 接入HLS实时视频流，支持全屏查看。
  - **AI识别结果**: 自动抓拍并识别鱼类病害（烂鳃、赤皮、肠炎），红色框标注病害个体。
  - **KPI指标**: 显示当前存栏预估、平均体重等关键生产数据。

## 2. 水质监测详情
- 提供24小时历史趋势图表，帮助分析水质变化规律。
- 支持按时间范围查询历史记录并导出为Excel报表。

## 3. 鱼类病害识别
- **实时识别**: 在视频监控过程中自动进行AI识别。
- **手动上传**: 支持上传本地抓拍图片进行深度识别。
- **置信度调节**: 用户可根据需求调整识别置信度阈值（建议50%-80%）。

## 4. 精准投喂管理
- **参数设置**: 设置饲料系数、投喂频次和粒径。
- **模糊PID**: 系统预留模糊PID算法接口，后续可实现基于水质和鱼类状态的自动投喂量优化。
- **手动操作**: 支持紧急情况下的手动投喂指令下发。

## 5. 系统管理
- **用户权限**: 超级管理员可配置操作员账号。
- **系统设置**: 支持亮色/暗色主题切换，适配不同办公环境。
